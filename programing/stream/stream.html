Streamとは

LinuxPCのメモリ帯域幅測定に使用する、Streamについて少し触れていく。

STREAM

Sustainable Memory Bandwidth in High Performance Computers

cs.virginia.edu

使用方法

私はUbuntuのPCで使用してみました。

STREAMのソースコード
cs.virginia.edu

stream.cを持ってくる。/streamディレクトリ内に保存する。

/stream

stream.cをコンパイルする。

bash

gcc -O stream.c -o streamsingle

streamsingleを実行する。

bash

./streamsingle
